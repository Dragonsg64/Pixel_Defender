Pixel Defender


Description:

	C'est un jeu dans le thème du tower défense.
L'objectif d'un tower défense est de protéger la base des monstres qui l'attaque.
venez jouer à ce tower défense pour passer le temps en vous amusant


Installation:

	pour installer ce projet il suffit de faire un git clone du jeu
puis de démarrer le fichier "Pixel_Defender - Raccourci.bat".
dans le cas où cela ne démarre pas la page de connexion, relancer le fichier


Utilisation:

	Pour commencer créez-vous un compte, ensuite connectez-vous et démarrait le jeu
une fois dans le jeu vous pouvez lancer une partie en cliquant sur le bouton start au milieu de l'écran
après cela la partie commence.
